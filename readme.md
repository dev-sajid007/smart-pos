## About Retail

A Simple user friendly retail managemnent application including,

- **Sales Management**
- **Purchase Management**
- **Due Payment**
- **Advance Receive**
- **Stock Management**
- **Quotation Management**
- **General Account**
- **Customer, Supplier, Vendor**
- **SMS Service**
- **MIS Report**
- **Access Control Management**
